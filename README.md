# Classifying Heartbeats and Detecting Arrhythmia (Machine Learning Approach)
_This repository is made to collaborate on creating the scripts and architectures used to complete this project._

## Prerequisites
- Python 3.13.x or later version [Download Python](https://www.python.org/downloads/)
- A copy of the MIT-BIH Arrhythmia database [Download here](https://physionet.org/content/mitdb/1.0.0/)
_Download the zip file; extract it into a folder and name it "mit-bih" and place it in the root directory of the repository for the scripts to behave expectedly._
